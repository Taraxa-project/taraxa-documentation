# Install a Node

{% hint style="info" %}
Installing a node takes only a few minutes. _The easiest way to deploy a persistent node that will be "always-on" is using a VPS service in the cloud._
{% endhint %}

## Minimal requirements

* CPU: 4vCPU
* RAM: 8 GB
* DISK SPACE: 50 GB
